Проектирование и изготовление пресс-форм и штампов

Изготавливаем конструкций следующих видов формовой оснастки:

  • Пресс-формы для литья термопластов;
  • Пресс-формы для прессования  реактопластов;
  • Пресс-формы для прессования резиновых  смесей;
  • Пресс-формы для литья алюминиевых сплавов;
  • Формы для литья  по выплавляемым моделям;
  • Формы для заливочных компаундов и смол;
  • Кокили для литья цветных металлов;
  • Фильеры;
При разработке проекта мы тщательно изучаем совместно с заказчиком условия работы оснастки, обрабатываемый материал, ресурс изделий, качество поверхности. По модели изделия, изготавливается технологическая оснастка. Используемое CAD программное обеспечение, учитывает температурное расширение оснастки и изделия в процессе работы и корректирует размеры оснастки для получения точных размеров.  После изготовления оснастки проводятся пробные испытания на заводе с измерением получаемой геометрии детали. При необходимости оснастка корректируется с повторными испытаниями.
Наша компания непрерывно отслеживает новые тенденции в области материаловедения, новых методов механической обработки металла и термической обработки металлов, а также нанесение функциональных вакуумных покрытий методами PVD технологий (износостойкие и антиадгезионные свойства). Отработана и внедрена в производство нанесение покрытий из нитрида хрома (CrxNу), методом HIPIMS, позволяющая увеличить ресурс оснастки в несколько раз.

Все вышеперечисленное позволяет выполнять самые сложные задачи по изготовлениютехнологической оснастки в кратчайшие сроки.
